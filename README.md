# claude-plugins

A personal Claude Code plugin marketplace by [kudima03](https://github.com/kudima03).

## Register this marketplace

```
/plugin marketplace add https://github.com/kudima03/claude-plugins
```

## Install plugins

```
/plugin install nuget-publish-skill@kudima03
```

## Plugins

| Name | Description |
|---|---|
| [nuget-publish-skill](https://github.com/kudima03/nuget-publish-skill) | Full NuGet publish workflow: version analysis, API compat check, suppressions PR, tag push, baseline update |
